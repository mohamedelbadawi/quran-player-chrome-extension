# Wzaker

it's a chrome extension that let the user to listen to all Qura'an swar by his favourite Qura'an reader


## Sheiks  list
- Abdulbaset Abdelsamad
- Yasser Eldossari

## Functions
- choose swrah
- choose sheik
- Random swrah
- Previous swrah
- Next swrah
- Repeat swrah
